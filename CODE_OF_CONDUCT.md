# 行動規範（Code of Conduct）

## 私たちの約束

Nivaの想映コミュニティの全メンバーとして、私たちは以下のことを約束します：

- 年齢、体型、障害、民族性、性的同一性と表現、経験レベル、国籍、外見、人種、宗教、性的同一性に関わらず、すべての人に対して尊重と敬意を示す環境を作成する
- 建設的で、包括的で、安全なコミュニティを構築する
- 多様な視点や経験を歓迎し、尊重する
- 建設的なフィードバックを受け入れ、提供する

## 私たちの基準

### 受け入れられる行動

以下のような行動は、健全で建設的なコミュニティを作成するために推奨されます：

- 他のコミュニティメンバーに対して敬意と思いやりを示す
- 異なる意見や視点を尊重する
- 建設的なフィードバックを与え、受け入れる
- コミュニティの利益のために行動する
- 他のコミュニティメンバーを尊重する

### 受け入れられない行動

以下のような行動は許容されません：

- 嫌がらせ、差別、または脅迫的な言動
- 個人的な攻撃や侮辱的なコメント
- 他のメンバーのプライバシーやセキュリティの侵害
- スパム、宣伝、または不適切なコンテンツの投稿
- その他の不適切または有害な行動

## 実施

### 違反の報告

コミュニティガイドラインの違反を目撃した場合は、以下の方法で報告してください：

- **GitHub Issues**: 違反を詳細に説明して GitHub Issues で報告
- **メール**: プロジェクトメンテナーに直接メール
- **公式 LINE**: https://lin.ee/nJQQAw41

### 対応

違反が報告された場合、プロジェクトメンテナーは以下の対応を取ります：

1. **調査**: 報告内容を調査します
2. **対話**: 関係者と対話します
3. **判断**: 違反の程度に応じて判断します
4. **対応**: 必要に応じて措置を講じます

### 措置

違反の程度に応じて、以下のような措置が講じられる可能性があります：

- **警告**: 初回の軽微な違反に対する警告
- **一時的な制限**: コミュニティへのアクセスを一時的に制限
- **永久的な除外**: 重大な違反に対するコミュニティからの永久除外

## 適用範囲

この行動規範は、以下の場面で適用されます：

- GitHub リポジトリ（Issues、Pull Requests、Discussions など）
- プロジェクト関連のコミュニケーションチャネル
- プロジェクト関連のイベントやミートアップ
- その他のプロジェクト関連の活動

## 帰属

この行動規範は、[Contributor Covenant](https://www.contributor-covenant.org/) v2.0 に基づいています。

## ご質問

この行動規範についてご質問がある場合は、プロジェクトメンテナーにお問い合わせください。
